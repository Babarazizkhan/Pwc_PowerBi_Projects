# Project Title: PwC Switzerland Power BI Project
# Overview
This project, completed as part of the PwC Switzerland Power BI Project, focused on enhancing my skills in data visualization and analysis using Power BI. In this simulation, I worked on creating dashboards to effectively communicate key performance indicators (KPIs) and provide actionable insights to clients. Additionally, I addressed gender-related KPIs in HR data, identifying underlying causes for gender balance issues at the executive management level.

# Key Highlights:
    Power BI Dashboards: Created clear, actionable visualizations of KPIs, showcasing the ability to respond to client needs with well-designed solutions.
    Data Analysis: Analyzed HR data, focusing on gender-related KPIs, to identify root causes for gender balance issues in executive management.
    Communication Skills: Delivered concise and informative email communications with engagement partners, providing valuable insights and suggestions based on the data analysis.
    Problem-Solving: Applied analytical thinking to examine the data, demonstrating a strong commitment to data-driven decision-making and client satisfaction.
# Tools Used:
     Power BI: For creating interactive dashboards and visualizing key metrics.
     Excel/HR Data: Used for data analysis and identification of gender balance issues.
# Conclusion
This project helped me strengthen my Power BI skills and my ability to communicate data-driven insights clearly. It also demonstrated my ability to analyze complex data sets, solve problems, and deliver valuable recommendations to clients in a business environment
